Module 01

Comprendre les Cryptos

Le guide essentiel pour maîtriser les fondamentaux de la blockchain et naviguer dans l'écosystème Web3.

🔗

Qu'est-ce qu'une blockchain ?

Imaginez un grand livre comptable numérique partagé entre des milliers d'ordinateurs. Chaque transaction est inscrite de manière indélébile.

L'Analogie

Comme un groupe WhatsApp où personne ne peut supprimer ses messages : tout est public et permanent.

La Puissance

Pas besoin de banque centrale. C'est le réseau lui-même qui vérifie et sécurise les échanges.

💰

Qu'est-ce qu'une crypto-monnaie ?

Une unité de valeur numérique qui sert de carburant (Ethereum) ou de réserve de valeur (Bitcoin).

BITCOIN (BTC) — L'or numérique
ETHEREUM (ETH) — L'ordinateur mondial
👛

Les Wallets

Votre porte d'entrée dans le Web3.

🌐

Adresse Publique

Votre IBAN crypto. Partagez-la pour recevoir des fonds.

🔑

Clé Privée

Votre code secret. Ne jamais la partager. Qui possède la clé possède les fonds.

⚖️

CEX vs DEX

Où échanger vos cryptos ?

CEX

Centralisé (ex : Binance). Facile à utiliser, mais ils gardent vos clés.

DEX

Décentralisé (ex : Uniswap). Vous gardez le contrôle total via votre wallet.

🏗️

Layer 1 vs Layer 2

Le Layer 1 est la base (Ethereum). Le Layer 2 (Arbitrum, Optimism) est une autoroute construite au-dessus pour aller plus vite et moins cher.

⛽

01
